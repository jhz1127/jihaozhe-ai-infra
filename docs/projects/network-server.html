高性能 C++ 分布式网络服务器框架

基于非阻塞 I/O 和多 Reactor 模型的事件驱动通信框架。

个人项目 · 独立开发 · C++11 / Linux / Epoll / Multi-Reactor / RAII

设计目标

构建可扩展的非阻塞网络服务器，处理连接管理、事件分发、跨线程回调与对象生命周期，并尽可能减少 I/O 过程中的系统调用开销。

架构图

Main Reactor

accept 新连接、分发 EventLoop

→
Sub Reactor

epoll 事件循环、线程池执行

→
TcpConnection

读写 Buffer、回调与生命周期管理

关键实现

  • 基于 One Loop Per Thread 实现 Main-Sub Reactor，多线程并行处理 I/O 事件。
  • 封装 Channel、EventLoop、TcpConnection 等核心组件，使事件注册、分发与连接状态解耦。
  • 设计应用层 Buffer，结合分散读减少系统调用；以 shared_ptr/weak_ptr 管理跨线程回调对象的生命周期。

工程收获

项目系统性实践了 Linux 网络编程、事件驱动设计、线程间协作与 RAII 资源管理，为后续的分布式训练和高性能数据链路工作建立了底层系统基础。
