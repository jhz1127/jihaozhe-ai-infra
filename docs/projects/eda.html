EDA 精英挑战赛：OpenMP + SIMD 两级并行

海量版图多边形模板匹配与聚类优化。

个人项目 / 竞赛 · 队长 / 第一完成人 · C++17 / OpenMP / NEON SIMD

问题

数十万个版图需要 O(N²) 成对几何相似度计算。赛题限制 90 秒，单线程基线超过 40 分钟；不同版图对复杂度差异大，固定分配会造成严重拖尾。

架构图

算法层

多级剪枝、集合覆盖、局部 / 禁忌搜索

线程层

OpenMP dynamic schedule，64 核负载均衡

数据层

AoS → SoA，NEON 向量化与对齐加载

关键设计

  • 使用 schedule(dynamic) 让线程按完成速度领取下一版图对，单项消除拖尾加速 1.8×。
  • 将点存储由 struct Point{x,y} 改为 xs[N], ys[N]，使编译器在 ARMv8 上生成高效 NEON 向量化代码。
  • 以多级几何剪枝缩减搜索空间，并把版图聚类建模为集合覆盖，采用局部搜索和禁忌搜索迭代优化。

结果

全国二等奖队长 / 第一完成人
10×+相对基线加速
−50%+聚类数量减少，90 秒内完成
