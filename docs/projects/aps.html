APS 高级计划与排程系统（校企合作）

面向多级生产批量计划的多约束组合优化系统。

个人项目 · 核心开发者 · C++20 / CMake / MLCLSP / 启发式算法

问题

多级生产批量计划（MLCLSP，NP-Hard）需同时处理机器产能、模具组合、BOM 层级、批次生命周期、班次成本与替代规则等十余种约束。商用求解器在多数算例上难以在实际时间预算内获得高质量可行解。

关键设计

  • Attributes / State 分离：将不变输入与可变排程状态拆开，支持全状态快照、增量修改与违规回滚。
  • FlatSet 模具匹配：以产品特征为 key 直接索引兼容模具，将高频 O(n) 遍历改为近似 O(1) 查找。
  • Chain Tracking：跨天追踪深层 BOM 依赖链，确保子部件先于父部件排产；配合滚动窗口处理新订单插入。
