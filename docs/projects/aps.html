APS 智能排程系统

以领域约束建模和启发式搜索解决多层 BOM、模具与产能约束下的复杂排程问题。

个人项目 / 竞赛 · C++20 / 组合优化 / MLCLSP

问题与目标

工业订单同时受到产品交期、设备产能、模具替代、班次和多层 BOM 依赖限制。通用求解器难以直接表达业务规则，且订单动态插入时重算代价高。我构建了可回滚的状态模型与分阶段启发式算法，在保证约束零违规的前提下追求更好的生产目标。

静态规则、可变排程状态与启发式求解解耦，支持在局部变更后快速回滚和重排。

关键设计

  • Attributes / State 分离：不变约束与每天的排程结果分开保存，使候选方案可以全状态快照、增量修改和安全回滚。
  • 领域剪枝：用 FlatSet 加速模具匹配，用替代规则和生命周期先排除不可能的候选，减少搜索空间。
  • Chain Tracking：显式追踪跨天 BOM 链，避免深层物料依赖在局部排程中被遗漏。
  • 滚动窗口：面对新订单，不从零求解；在有限窗口内渐进重排，兼顾响应速度与方案质量。
