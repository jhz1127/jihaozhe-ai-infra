PipeProbe：DataLoader Pipeline 低侵入诊断

让诊断数据随真实 batch 回流，精确定位数据管线的生产、排队与消费瓶颈。

京东实习项目 · Python / PyTorch DataLoader / multiprocessing

问题

自定义 DataLoader 包含 read、parse、tokenize、collate 等阶段。传统旁路 profiler 需要额外 IPC 队列和聚合线程，容易与真实 batch 错位并产生额外开销。

关键设计

  • 将 _TimedBatch 通过 PyTorch 已有数据通道返回，保持诊断信息和真实 batch 严格对齐，实现零额外 IPC。
  • Span 支持嵌套并计算 exclusive time，可将任意业务代码块映射为独立耗时阶段。
  • 并行观测 worker wall、queue dwell、fetch wait 和 train-step cadence，按 worker 输出 P50/P90/P99。

方法论

全局均值只能说明“存在问题”；按 worker / rank 展开分布才可定位异常个体。生产、传输、消费三段对齐是识别数据管线瓶颈的基础。
